1.1 Introducción a las aplicaciones

1.1.1 Direccionamiento

  • Direccionamiento Transporte (TCP/UDP)

    Para identificar al proceso de la aplicación hay dos aspecto de información:

    • Nombre o dirección del host (el host se identifica mediante su IP)
    • Identificador que se especifique el proceso emisor/receptor en el host origen o destino. Se llama número de puerto de nivel de transporte.

    El número de puerto de destino sirve identificar una de estas posibles  aplicaciones.

    Las aplicaciones populares tienen asignados números de puerto específicos. Por ejemplo, un  servidor web queda identificado por el número de puerto 80.

  • Direccionamiento de los procesos de aplicación

    Un proceso envía mensaje a la red y los recibe de la red a traves de una interfaz SW denominada socket. La pareja de datos: Dirección IP: puerto identifica unívocamente un socket.
